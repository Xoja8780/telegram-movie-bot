# 🎬 Telegram Movie Bot

A Telegram bot built with Python and aiogram that allows users to browse movies, view details, and save them for later.

## 🚀 Features

* Browse movies by genres
* View movie details (title, description, rating)
* Watch trailers
* Save movies to "Watch Later"
* Database integration

## 🛠 Tech Stack

* Python
* Aiogram
* SQLite
* TMDB API
* dotenv

## ⚙️ Setup

1. Clone the repository:
   git clone https://github.com/yourusername/telegram-movie-bot.git

2. Install dependencies:
   pip install -r requirements.txt

3. Create .env file:
   TELEGRAM_TOKEN=your_token
   API_KEY=your_tmdb_key

4. Run the bot:
   python main.py

## 📌 Project Purpose

This project was built for learning backend development and working with APIs, databases, and Telegram bots.
