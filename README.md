# 🎬 Telegram Movie Bot

A Telegram bot built with Python and aiogram that allows users to browse movies by genre, view details, and save movies to a "Watch Later" list.

## 🚀 Features

* Browse movies by genres
* View movie details
* "Watch Later" functionality
* Database integration
* TMDB API usage

## 🛠 Tech Stack

* Python
* Aiogram
* SQLite / PostgreSQL
* TMDB API

## 📌 How to run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Add your .env file with API keys
4. Run:
   python bot.py
